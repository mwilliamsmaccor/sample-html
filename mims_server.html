MIMS Server

Discover the immense power of Maccor's Information Management System (MIMS) Server. The MIMS Server automatically creates system file backups and generates indexed data files for easy viewing with the MIMS Client. By centralizing data from multiple systems, you can save time and effort when analyzing your results. Additionally, you have the option to store the data in standard ASCII format.

Set It and Forget It

Once configured, the Server runs in the background leaving you free to perform other tasks

Output

Configure the MIMS Server to automatically output an ASCII file from all the connected testers. Select the desired fields available and the file options pertinent to your needs.

Synchronize

Designed to reduce errors and maintain uniformity across all of your Maccor testers, the MIMS Server can be configured to automatically synchronize test procedures on all the Client PC's from a master location.
