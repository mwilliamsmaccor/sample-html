M22 Virtual Machine Solution

The M22 Virtual Machine Solution frees up lab space, increases reliability, and optimizes large scale installations efficiency. With one M22 you can replace 22 desktops in your laboratory. The M22 shared reduntant drives to ensure data and system integrity. The hyper-fast internal bus integrates perfectly with Maccor's MIMS  Server ensuring up to date data backups.
